Forms and JS

Form Ex1

Create a web form that collects a name, age, and favorite color. For the color lets restrict it to be only a few colors. Use an approriate for element to do so (ex: restrict it so they can only pick red, green, and blue).

Be sure to add a button that allows them to submit. Have to form use POST. When a form is eventually submitted make sure it goes to a page called process.php in this same directory. We will work on this page later so you don't need to upload all your work to the webserver yet.

Form Ex2
